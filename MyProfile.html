Project Idea

When faced with two types of trash bins, people are often confused by what type of waste is considered landfill and what is recyclable. The compact description of items belonging to each bin provided in the area is also often limited. This project involves the development of an application that is intended to help users manage their waste. The application will enable users to decide whether their waste is recyclable in a few easy steps. Users can search for their waste on the search bar in the application, which will then tell them whether it will be treated as landfill, or recyclable.

Australia is a big recycler, 60% of the total waste produced in 2014 to 2015 were recycled. Therefore, there is a high demand for products that encourage and provide better education on recycling. The government has also emphasized on the importance of correctly sorting garbage before throwing them. This is to avoid further crisis that has been going on with the Australian recycling industry recently, where recyclables ended up in landfill. BinMe is a simple solution for people who want to start recycling.

The application will feature a welcome screen showing the name of the application; “BinMe” and its logo, which will be a tree. This interface will be mostly green in colour and is a loading screen before the next one. The next interface will include a search bar in the middle of the screen and “BinMe” on top of it.

After entering the keyword, users will be able to press on either “Go” on the right hand bottom corner of their keyboard or the magnifying glass symbol on the search bar itself. Both the “Go” key and the magnifying glass symbol will bring users to the next page, which will include a list of all matching words. For example, if the user searches for “tissue”, the list will include items such as “tissue paper”, “tissue box” and “wet tissue”. This second page will also feature a back button symbolized by “<--” that will bring users back the search bar page when tapped.

Users will then select one of the items from the list by tapping on the name of the item. If the chosen item is recyclable, the application will proceed to show a screen with the name of the item on top of the page, followed by the word “Recyclable”, a yellow tick sign and a trash bin with a yellow lid. The chosen colour reflects the yellow recycling bins around Melbourne.

On the other hand, if the chosen item is considered to be general waste, the application will show a page with the name of the item on the top, followed by the word “Landfill”, a red cross sign and a trash bin with a red lid. Similarly, this reflects the colour of landfill waste bins in Melbourne.

Both the “Recyclable” and “Landfill” interface will also feature a back (<--) button that will bring users back to the search page. In order to have a complete list of items, the application will be equipped with a database system that consists of an extensive list of everyday waste. Although by no means exhaustive, this list will cover most items in our daily waste. This is a significant feature of the application since not having the item they are looking for will most likely result in users throwing their waste in the general waste bin regardless of whether it is recyclable. The application will also feature a database system that stores the information of whether something is recyclable or not. Since this particular database needs to be extremely accurate, there will be a lot of research that needs to be done in this area.

Software needed to create this application is called Appery.io. It is a mobile application builder that can create apps for both Android and IOS. The application has a built in visual editor with a drag and drop feature which will auto-generate codes for each component. Furthermore, it will be able to connect to any REST API, which will add a cloud database onto the application. Equipment needed includes hardware such as computer and a smartphone. The computer will be used for designing and developing the application while the smartphone will be needed for trial runs of the application.

Skills that are required for this project are knowing the basics of UX design, which can be done through observing the targeted users and taking on their perspective. It is very feasible to find the software required to design the application as Appery.io is an online platform that is accessible to anyone. Although it is a paid service, it is guaranteed to ease the development process of this application.

Upon the completion of the project, the problem of not knowing whether something belongs in general waste or mixed recycling bin will be solved. Users will be able to check whether their waste is recyclable very quickly and without any hassle. The application will positively impact the recycling industry in Australia by helping users correctly sort their waste, and the collection company who will have an easier time recycling waste that are actually meant to be recycled.
